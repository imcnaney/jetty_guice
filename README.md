# jetty_guice
Java server with embedded Jetty and Guice dependency injection.

# Basics
see OEServer.java to see how it all ties together.

# To Run
`./gradlew runShadow`

Hit http://localhost:8100/ for the one and only API call, and to get a welcome message.
